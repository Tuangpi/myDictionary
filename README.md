# myDictionary

`myDictionary` is a small front-end dictionary app built with plain HTML, CSS, and JavaScript.

It lets users search for words, view simple meanings, see autocomplete suggestions, and keep a recent search history in the browser using `localStorage`.

## Features

- Search for a word and show its meaning
- Autocomplete suggestions while typing
- Recent search history saved in the browser
- About, Settings, and Acknowledgement views
- Basic mobile-friendly navigation

## Tech Stack

- `index.html`
- `style.css`
- `main.js`

No build tools or frameworks are required.

## Project Structure

- `index.html` - app layout and sections
- `style.css` - styling and responsive layout
- `main.js` - dictionary data and app logic
- `icon-hamburger.svg` - mobile menu icon

## How to Run

1. Clone or download the project.
2. Open `index.html` in your browser.

You can also run it with a simple local server if you prefer.

## Notes

- The dictionary data is currently hardcoded in `main.js`.
- Recent searches are stored in the browser under `localStorage`.
- This project is fully client-side.
